第十二章二：渗透 - 5

本森的马车在通过外门的基础检查后，悄无声息地驶入营地。由于本森对此地极为熟悉，加之这种例行送货本就寻常，守卫们甚至只多瞥了一眼便放行了。

车轮碾过石板地面，发出沉闷的滚动声。马车缓缓驶入这处不大的营地，威廉便从隐蔽处冷静地做好了行动的准备。

他取出了武装·纽克斯，将其重塑为【赤色永劫】，并立刻激活了其隐身特性。因为他深知，一旦开始移动，那件法袍反倒会成为累赘而非助力。

尽管法袍的隐身效果远胜于【赤色永劫】，但它在战斗中对动作的限制太过严苛。威廉至少明白，一旦鲜血开始流淌，机动性远比隐匿更为珍贵。

马车停稳在主厅入口前，几名身着与本森相似黑袍的男子不待片刻便赶了过来，开始将那些被锁链束缚的俘虏一个个卸下，动作粗鲁地将其们押往地下室。

威廉在自身隐身的效果之上，悄然开启了【蚀日面纱】。虽然两种效果重叠，但【蚀日面纱】能在一定程度上压制声响，从而减少他此刻移动时的噪音。

跟随本森进入主厅后，威廉观察着内部景象，发现人们神色慌张地四处走动，窃窃私语，仿佛出了什么大乱子。

“表现得自然些，去跟个人聊聊，”威廉通过奴隶印记向本森下令道，“问问他们在搞什么鬼。”

本森立刻领命，走向附近的一名教众。与此同时，威廉抬起目光望向天花板，注意到上方的结构是由金属横梁与木质支撑构成的开放式桁架系统。

威廉便沿着支撑杆悄然攀爬，将自己定位在主厅上方，因为下方实在太过拥挤。

从屋顶俯瞰，他既能观察一切，又不会轻易撞到人而暴露行踪。

从这个制高点，他很快便理清了局势。

“我的主人，”本森的声音在他脑海中响起，“那头兽人公主竟然在营地内鬼鬼祟祟地溜达。”

威廉在乌鸦面具下嘴角微微上扬，发出一声低沉的轻笑，随即在意识中回应道：

“看来我所料之事，终究成真了。”

“我真是个天才。”威廉在意识中带着过分的自信自夸道。

刹那间，命运似乎真的站在了他这一边。

然而，仿佛命运本身就在等待他吐出那句自夸，随即便决定与他作对，证明他大错特错。

轰！！

一道恶魔火球术径直轰击在他方才栖身的屋顶区域，木梁与金属一同碎裂，火焰向外猛烈爆发。

屋顶结构瞬间坍塌，威廉的身影在最后一刻于半空中灵巧闪避，身体顺势扭转，任由碎屑如雨般从头顶洒落。

下方站着一名黑袍教众，双手仍泛着刚刚释放法术后的微光。

威廉倒也不担心暴露身份，毕竟乌鸦面具与一身漆黑的装束已将他隐藏得严严实实，真正让他在意的是，此人究竟是如何感知到他的存在的。

那教众不过 SS 级。

“那就亲自问问他，”威廉在意识中冷静地决定道。

他再次抬眼，注意到另一个存在正从另一个方向急速逼近，一头龙裔的剪影在天空背景下浮现，双翼展开，仅数秒便悬浮于他面前。

龙裔感知到威廉的等级后，嘴角轻蔑地勾起，言语间满是毫不掩饰的 disdain。

“啧，又一个麻烦。”

威廉缓缓举起长剑，用一种沙哑而扭曲的声音回应道：

“你刚才说了什么？”他压低嗓音问道，“我赌你在一分钟内能说出同样的话。”

龙裔发出嘲弄的笑声。

“嘿嘿，你这乌鸦打算拿我怎样？”

“拍脸，”威廉低笑一声，随即在身侧开启了一道门户。

“陪他玩上一分钟，太阳。”

“遵命，主人。”

【追日】瞬间现身，庞大的身躯遮蔽了天空，随即毫无等待地向着龙裔扑去，战斗就此爆发。

<< 5 秒 >>

威廉并未回头观看这场战斗。

他反而冷静地俯冲而下，将本森召回【无尽领域】，毫不延迟地将其从战场上撤下。

<< 10 秒 >>

威廉出现在聚集的教众中央。这些人早已拔剑出鞘、准备施法，脸上写满了恐惧与困惑。

“虚空君权术，第一式：空间锁。”

“赤色永劫·恶魔龙化。”

“星炎。”

“流光步。”

<< 12 秒 >>

【赤色永劫】的恶魔形态剧烈震颤，那裸露的金属獠牙相互咬合、摩擦，发出蓄势待发的声响，仿佛武器本身也在渴望鲜血。

威廉的身影爆发出 blazing 的动感，在教众们甚至来不及感知疼痛的瞬间，他们的身体便开始四分五裂。

滋啦！！

威廉如同一台活生生的研磨机，撕裂着整个基地，将那些被【空间锁】禁锢、动弹不得的冻结身影一一切开。

一百三十多名教众被定在半空，脸上凝固着极致的惊恐。

威廉的动作带着 relentless 的动量，连绵不绝。

与此同时，【赤色永劫】的切割并不干脆。

它的獠牙碾碎骨骼、撕裂皮肉、绞断筋脉、崩断肌腱，最后才切断肢体，将每一次挥击都变成了 prolonged 的酷刑，而非 swift 的利刃。

这种体验就像锯子切割肢体，比起干净利落的斩击，它更加痛苦且缓慢。

受害者们只经历了一瞬致盲的闪光，紧接着便是令人毛骨悚然的碎裂声，随后剧痛在体内炸开，器官在剑刃锋利的獠牙下暴力破裂。

<< 20 秒 >>

有些人自认幸运，以为仅断肢便已万幸，至少能活到首领来处置这个疯子，但他们的希望很快便化为泡影。

【星炎】立刻感染了他们的伤口，如同一种恶魔能量的感染，完全阻止了再生。

星炎让他们感到一种直抵灵魂的剧痛与折磨，尖啸声穿透了空气。

威廉并未停歇。

他从一处尸群移动到另一处，无情地将身体劈开，伴随着此起彼伏的惨叫声，整个崩塌的基地回荡着死亡的乐章。

<< 30 秒 >>

少数人试图抵抗，但他们的身体仿佛被某种无形的空间约束所禁锢，无法动弹，那是他们无法 comprehend 的束缚。

“起源断绝。”

威廉的声音冷冷地响起，一道巨大的剑能弧线轰击在木质地板上，将身下的结构彻底压垮，与此同时，他也解除了【空间锁】。

剩余的教众随着地板塌陷而向下坠落，露出了三层地下结构，最底层显然被 designate 为囚禁区。

威廉悬浮于他们之上，双眼燃烧着 unrestrained 的专注。

他再次俯冲而下，宛如一名行刑者，穿梭于坠落的碎屑与尖叫的尸体之间，将肉体雕琢得如同 raw material 一般，而非在屠杀活物。

上方的龙裔仍与【追日】缠斗，并未目睹下方的屠杀。

威廉的表情扭曲成了某种难以辨认的模样，仿佛他正在创作艺术品，而非进行杀戮。

【空间锁】再次激活，将那些试图逃跑的教众定在半空，定格在冲刺的瞬间。

随着被摧毁的结构，碎屑纷纷升腾而起，威廉开启了真武装·臣服艺术——七十二手控。

巨大的石块、金属与木块围绕着他漂浮，他缓缓抬起双手，将它们高高抛向天空。

他给这些碎片涂上了火元素。

手腕轻轻一抖，这些燃烧的碎屑便如地狱般的流星雨般逐一降下。

那些被直接命中的目标瞬间化为浆糊，活生生的身体在撞击中爆裂开来，每一次碰撞都让更多的地下结构随之坍塌。

七十二块碎片无情落下，几乎屠戮了所有人。

<< 45 秒 >>

当破坏终于平息，威廉再次冷静地俯冲而下。

只有一处区域完好无损。

囚犯们被刻意移开，此刻被一道由君权剑意构成的屏障所包裹，未受丝毫浩劫的波及。

另一道独立的屏障保护着那位早些时候袭击威廉的 SS 级教众。

威廉悬浮向那名颤抖的男子，他早已失禁，完全失去了控制。

“你是如何感知到我的？”威廉用沙哑而冰冷的声音问道。

<< 50 秒 >>

那人仍僵在震惊之中。

“说！！”威廉厉声喝道。

“是加热法阵出现了干扰，”那人结结巴巴地回答，“你挡住了热气。”

威廉皱起了眉头。

“该死？明明并不冷，为何需要加热法阵？”

“那里很冷，”那人紧张地咽了口唾沫。

威廉立刻执行了处决，不再对这种无趣的废话感兴趣。

<< 滴答滴答…… 55 秒 >>

威廉出现在龙裔面前，【追日】则听令退至他身侧。

威廉指向下方，用一副得意的口吻说道：

“往下看，你这卑鄙的蜥蜴。”
