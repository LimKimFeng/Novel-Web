第二十一章 调查（一）

威尔缓缓醒来，首先感觉到的便是颅内隐隐作痛。

那痛楚并非尖锐，而是持续不断，仿佛有什么东西从内部紧紧挤压着他的头颅。喉咙干涩，四肢僵硬，让他觉得自己昏迷的时间远比记忆中漫长得多。

他站起身，抓起一只水罐，仰头将整罐水一饮而尽。

“唉……这已经是这周第二次晕倒了，”他不满地嘟囔道。

他的声音显得虚弱而沙哑，停顿片刻后才再次尝试活动身体。

他完全睁开双眼，凝视着上方的天花板。那由苍白的石块砌成，几道细微的裂纹纵横其间，显然年久失修，维护不善。

四角刻着小小的法阵，本意是维持房间的无菌环境以防感染，但此刻光芒微弱且闪烁不均。

空气中弥漫着药香，混合着草药、消毒水和陈旧布料的异味，让他确切地知晓自己身在何处。

“公共医务室，”威尔低语道，“果然。我真是惹了一身骚。”

他微微侧头，打量着四周。病房内排列着成排的病床，每张床上都躺着一位受伤平民。

有人睡得不安稳，有人空洞地凝视着虚空，还有几人每挪动一下便发出轻微的呻吟。绷带缠绕在手臂、头部和躯干之上。

威尔小心翼翼地撑起身子，无视肌肉中蔓延的钝痛。四肢并未折断。

没有剧烈的内伤，只有过度透支魔力与身体极限后带来的疲惫与酸楚。

“系统，显示我的状态。”

【收到】

------------

姓名：威廉

种族：人类

年龄：14

排名：F+

天赋：无法无天（至高）

亲和：太阳（SSS级）

体魄：神日神体

功法：

——暴风剑术

——第一式（圆满）

——第二式（入门）

技能（关联体魄）：日冕威压、无尽日源、无限日愈、星焰、极寒克星、绝对日亲和。

技能（关联天赋）：绝对适应、命运免疫、无法无天觉醒、无限、万能语。

当前 SP：0

----------------

“天哪，”威尔苦笑，“若有人问我觉醒后一周便晋升了，他们会信吗？”

他终于明白了昨夜发生的一切。施展剑术耗尽了他体内的所有魔力，迫使魔力核心承受了巨大负荷。

通常情况下，魔力过用只会让人昏厥，不会有什么大碍，但在他身上，天赋却做出了反应。

【绝对适应强行扩张了你的魔力核心，不知为何，核心内部涌入了大量魔力，最终导致核心崩裂。】

【核心崩裂后，绝对适应再次生效，迅速适应并重塑了核心以容纳那庞大的魔力，最终促成了你的晋升。】

威廉叹了口气。常人修炼时也会使用所谓的“呼吸法”来扩张并 densify 魔力核心，从而逐步晋升。

“看来我得找一套修炼功法了，这方法既痛苦又危险。”

【其实也不至于那样，但既然昨晚那般拼命，还是尽量避免为好。】

“也是，你说得对，”威尔边说边用袖口擦去额头的汗水。

随即，他的思绪又飘回了那件事。

莉亚。

当他将双腿从床上垂下站起时，呼吸微微一滞。他迅速扫视着医务室，目光从一张病床移到另一张，检查着每一张面孔、每一具缠满绷带的躯体、每一个昏迷的患者。

她不在那里。

他的手指紧紧攥成拳头。

他再次搜寻，这次动作更慢，起初不愿接受这个事实。他走了几步，从床铺间隙中窥探，甚至掀开那些用于遮挡隐私的帘子一角。

一无所获。

一股寒意悄然沉入他的胸口。

“系统，”威尔压低声音，努力保持语调平稳，“她死了吗？”

【个体‘莉亚’已消耗宿主部分血液。是否使用 SP 追踪其位置？】

威尔皱起眉头，“可我身上没有 SP。”

【任务#1 已完成。】

【奖励发放：5000 SP。】

威尔稍作停顿，随即微微点头，“对。那就追踪她。”

【确认。】

【扣除 SP 100】

短暂的停顿后，威尔的耐心已濒临极限。

【个体‘莉亚’已被定位。】

【个体‘莉亚’生命体征稳定。】

【个体‘莉亚’尚存于世间。】

威尔缓缓吐出一口一直憋在胸中的气。

紧接着，下一行文字浮现。

【个体‘莉亚’已离开阿瑞斯大陆。】

威尔愣住了。

"……什么？”他问道。

【个体‘莉亚’已被传送至阿瑞斯大陆边界之外。】

【受宿主当前实力限制，确切位置难以确定。】

威尔揉了揉太阳穴，眉头紧锁，不安之感悄然蔓延。

“看来她并非自行逃走的，”他说道，

“而是被人带走了。”

【正确。】

他沉默片刻，消化着这些信息。思维飞速运转，重演昨夜的事件，并将其与记忆中小说的情节进行比对。

“系统，”他终于开口，

“你不觉得剧情现在完全偏离轨道了吗？”

回应即刻而至。

【建议宿主停止过度执着于原本的主线叙事。】

【正如先前所言，时间线 possess 自我修正的特性。】

【事件或许会变更，但因果律的大方向依然保持一致。】

威尔缓缓呼出一口气。

“你说得轻巧，”他低声嘟囔。

就在他想得更多之前，医务室入口处传来了脚步声。

“嘿，巧克力小子！”

威尔抬头，看见一张熟悉的面孔正朝他走来。一名中年男子，手臂上有一道疤痕，脸上挂着宽大的笑容，大步走来，脸上写满了如释重负。

“你终于醒了，”那人说道，“我还以为这辈子再也尝不到巧克力了。”

威尔立刻认出了他。这人是个老主顾， albeit 是个嗓门特别大的家伙。

“那倒真是悲剧，”威尔平淡地回应。

男子大笑起来，“少来这套。等会儿再给我打个折就行。”

随即，他凑近了些，语气略微压低。

“对了，这里有骑士。他们想问你几个问题。别担心，我告诉他们你是个卖甜点的无害小孩。”

威尔的肌肉微微绷紧。

“骑士，”他重复道。

“帝国骑士，”男子确认道。

威尔点点头，“多谢。”

待男子走远，威尔闭上双眼片刻。

他清楚接下来会发生什么。

“系统，”他在心中默念，“我需要一个撒谎的技能。”

【收到，既然免费特权已用尽，现在得用 SP 了。】

“知道了，”威尔回应道，“给我展示一个 SP 低于 1000 的技能。”

一块半透明的面板浮现在他面前，列出了一系列技能。威尔仔细浏览，从低阶能力到被动强化。

他的目光最终停留在一个条目上。

------------

虚假平静（B 级）（被动）

“真相在颤抖，而他的声音却稳如磐石。”

效果：

——施术者的面部表情、心跳、呼吸及微反应皆置于意识掌控之下，营造出完美的扑克脸。

——施术者所言的谎言，不会在标准测谎技能、测谎魔法、灵压读取或基于威压的审讯中产生可被察觉的情绪波动（对同阶或低阶技能有效）。

——施术者的心境保持冷静与平稳，即便在威吓、杀意或权威威压下，也能展现出超然的自信。

——抑制了紧张、犹豫与潜意识的愧疚反应，使言语流转如行云流水般自然顺畅。

备注：无法阻挡高阶的心灵读取、灵魂检视或直接的记忆提取。

价格：800 系统点

------------

威尔挑了挑眉，“这价格真便宜。”

【战斗类技能在商店中定价较高，但低廉的价格并不意味着效果不佳。】

“说得在理，”威尔说道，“买它。”

技能瞬间激活。

他几乎立刻感受到了变化。呼吸放缓，胸中的紧张感消散，思绪变得清晰而受控。

他的面容沉淀为一种自然的中性表情，而非刻意伪装。

一张完美的扑克脸。

“好了，”威尔低声说道，“该去应付这场麻烦了。”

不久，一名医官赶到，领着他穿过侧廊，进入医务室深处一间更小、更独立的房间。身后，门轻轻合上。

屋内，一名男子端坐在桌前。

他身材高大，肩宽背阔，举止间透着从容的自信。一把断剑斜背于身侧，并非刻意隐藏，而是常年佩戴的架势。

威尔一眼便认出了他。

布莱恩——剑圣克劳斯的右臂。
